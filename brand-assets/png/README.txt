Ninja Mascot Pack (SVG + PNG)
============================

Transparent background
----------------------
All files have no background and render transparent.

Consistent sizing
-----------------
Mascots: viewBox 0 0 200 200 (same framing across poses)
Icon:    viewBox 0 0 100 100

Dark mode / theming
-------------------
Outfit + outlines use 'currentColor'.
Set the CSS color on the wrapper to control the ninja tone.

Tailwind example:
  <div class="text-slate-950 dark:text-slate-100"> ... </div>

Files
-----
svg/  (SVG originals)
png/  (PNG renders 1024x1024)
